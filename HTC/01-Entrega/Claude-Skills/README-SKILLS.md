# HTC Claude Skills - Documentação Completa

## Visão Geral

Este repositório contém 10 Claude Skills profissionais criadas especificamente para o HTC (High Ticket Consulting). Cada skill foi desenvolvida com uma **estrutura robusta e modular** para automatizar e otimizar processos críticos de negócios High Ticket (ofertas de R$2k a R$50k).

---

## Nova Estrutura das Skills

A partir de 2026, cada skill segue uma **estrutura padronizada e expansível** projetada para oferecer documentação completa, exemplos práticos e fluxos de trabalho claros.

### Diagrama da Estrutura

```
nome-da-skill/

├── SKILL.md                     # Prompt principal (instruções para o Claude)

├── examples/                    # 5+ exemplos reais de uso
```

### Descrição dos Componentes

#### 1. README.md
**Propósito:** Visão geral rápida da skill
- O que a skill faz
- Quando usar
- Exemplo rápido de comando
- Link para documentação detalhada

#### 2. SKILL.md
**Propósito:** Prompt principal que orienta o Claude
- Instruções específicas para a IA
- Frameworks e metodologias
- Formato de saída esperado
- Regras e limitações

#### 3. MANUAL.md
**Propósito:** Instruções detalhadas de uso
- Como usar a skill passo a passo
- Parâmetros e opções disponíveis
- Dicas e melhores práticas
- Solução de problemas comuns

#### 4. examples/
**Propósito:** 5+ exemplos reais de uso
- Exemplos básicos para iniciantes
- Exemplos intermediários para usuários experientes
- Exemplos avançados para casos complexos
- Casos de sucesso reais
- Exemplos de erro e acerto

#### 5. templates/
**Propósito:** 3+ templates prontos para uso
- Template rápido (para resultados imediatos)
- Template completo (para resultados detalhados)
- Template específico (para casos particulares)
- Campos preenchíveis com [COLCHETES]

#### 6. references/
**Propósito:** Conceitos, frameworks, melhores práticas
- Conceitos fundamentais da área
- Frameworks utilizados
- Melhores práticas do mercado
- Casos de sucesso documentados
- Bibliografia e recursos adicionais

#### 7. workflows/
**Propósito:** Fluxos iniciante, intermediário, avançado
- **Workflow Iniciante:** Passos básicos para primeiros usuários
- **Workflow Intermediário:** Processos otimizados para usuários experientes
- **Workflow Avançado:** Técnicas avançadas para maximizar resultados

#### 8. prompts/
**Propósito:** Variações para diferentes contextos
- Prompts para webinars
- Prompts para calls de vendas
- Prompts para email marketing
- Prompts para redes sociais
- Prompts para diferentes nichos

#### 9. checklists/
**Propósito:** Para garantir qualidade
- Checklist pré-lançamento
- Checklist de qualidade de output
- Checklist de validação de resultados
- Checklist de otimização

## Como Instalar as Skills

### Método 1: Via Claude Code (Recomendado)

1. Baixe o arquivo .zip da skill desejada
2. No Claude Code, vá em **Settings** → **Skills**
3. Clique em **Upload Skill**
4. Selecione o arquivo .zip
5. A skill estará disponível imediatamente

### Método 2: Instalação Manual

1. Descompacte o arquivo .zip
2. Copie a pasta da skill para:
   - **Windows:** `C:\Users\[SeuUsuario]\.claude\skills\`
   - **Mac/Linux:** `~/.claude/skills/`
3. Reinicie o Claude Code

### Método 3: Instalação por Projeto

Para usar a skill apenas em um projeto específico:

1. Descompacte o arquivo .zip
2. Copie a pasta para: `[SeuProjeto]/.claude/skills/`
3. A skill estará disponível apenas neste projeto

## Melhores Práticas

### 1. Comece com Templates
Para resultados rápidos, comece usando os templates prontos em `templates/`. Eles já estão otimizados para funcionar.

### 2. Consulte Exemplos
Antes de criar algo do zero, consulte `examples/` para ver exemplos reais de uso.

### 3. Siga Workflows
Para processos completos, siga os workflows em `workflows/` para garantir que não esqueça nada.

### 4. Use Referências
Para entender melhor conceitos e frameworks, consulte `references/`.

### 5. Valide com Checklists
Antes de finalizar, use os `checklists/` para garantir qualidade.

### 6. Combine Contextos
Use os prompts em `prompts/` para adaptar a skill para diferentes contextos.

### 7. Seja Específico
Quanto mais específica sua instrução, melhor o resultado.

**Ruim:**
```
/htc-ofertas-creator Criar uma oferta
```

**Bom:**
```
/htc-ofertas-creator Usar template oferta-completa para programa de coaching executivo de 16 semanas, focado em diretores de vendas de empresas SaaS, preço R$15.000, incluindo 12 sessões 1:1, 4 group calls, material de apoio e acesso por 6 meses.
```
