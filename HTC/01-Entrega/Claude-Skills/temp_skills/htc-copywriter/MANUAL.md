# MANUAL DE USO - htc-copywriter

## Sobre Esta Skill

A **htc-copywriter** é especialista em copywriting High Ticket persuasivo. Ela cria copy para landing pages, emails, anúncios e scripts que convertem em vendas de R$3k a R$50k+, usando frameworks como CCDI, Armadilhas e Storytelling.

## Como Usar

### Sintaxe Básica
```
/htc-copywriter [tipo de copy] [solicitação]
```

## Perguntas Frequentes

**Q: Qual é a diferença entre copy curto e copy longo?**
A: Depende do estágio do funil:
- Topo de funil (frios): Copy curto, gatilho forte
- Meio de funil (mornos): Copy médio, mais contexto
- Fundo de funil (quentes): Copy longo, CCDI completo
- Página de vendas: Copy completo com todos os elementos

**Q: Como criar headlines magnéticos?**
A: Use framework "4 Us":
1. Útil: "Como [benefício] sem [dor]"
2. Urgente: "7 erros que te fazem [consequência]"
3. Único: "O método que [resultado único] em [tempo]"
4. Curiosidade: "O segredo que [promessa]"

**Q: Quantas CTAs devo incluir?**
A: Regra de ouro: 1 CTA principal no conteúdo. Em LPs, CTAs secundários ao longo do texto. Em emails, 1 CTA claro. Múltiplas CTAs confundem e diluem ação.

**Q: Como medir eficácia da copy?**
A: Métricas por formato:
- LP: Taxa de conversão, tempo na página, scroll depth
- Email: Taxa de abertura, cliques, respostas, conversão
- Anúncio: CTR, CPC, leads qualificados, custo por lead
- VSL: Retenção (25%, 50%, 75%, 100%), agendamentos gerados

**Q: Copy emocional ou racional?**
A: High Ticket precisa de ambos. Use CCDI:
1. Conexão (emocional)
2. Desapego (racional)
3. Descobrimento (racional)
4. Intensificação (emocional)
5. Oferta (racional com justificativa emocional)

**Q: Como tratar objeções na copy?**
A: Use "Prequeação" antes de oferecer:
- Antecipe objeções comuns
- Inclua seção FAQ dedicada
- Use prova social que neutraliza objeção
- Storytelling que supera objeção implicitamente

**Q: Qual é o comprimento ideal de LP?**
A: Para High Ticket:
- Versão curta: 1.500-2.000 palavras (mobile first)
- Versão longa: 3.000-5.000 palavras
- Foco: Não tamanho, sim clareza + persuasão
- Teste: Versão A vs B frequentemente

**Q: Como criar copy autoritária?**
A:
1. Use dados específicos ("234 alunos", não "centenas")
2. Mostre antes/depois com resultados
3. Inclua foto real e identificável
4. Conte vulnerabilidades que geram identificação
5. Use linguagem técnica simples, não jargão

**Q: Devo mencionar preço na copy?**
A: Depende do objetivo:
- Topo de funil: NUNCA mencione preço (gera clique, não venda)
- Meio/Fundo: Mencione âncora ("investimento a partir de 5 salários mínimos")
- Página de vendas: Preço claro com opções de parcelamento
- Sessão estratégica: Preço na hora certa (CCDI antes)

**Q: Como evitar copy genérica?**
A:
1. Especifique nicho ("nutricionistas para menopausa" não "mulheres")
2. Use nomes de armadilhas memoráveis
3. Inclua números concretos (exatamente, como, quanto)
4. Conte histórias reais (não "João Silva", mas "Maria, psicóloga de SP")
5. Seja você, não uma marca

## Troubleshooting

**Problema: Copy muito longa e tediosa**
- **Solução**: Use parágrafos curtos (2-3 frases)
- **Solução**: Adicione subtítulos e bullet points
- **Solução: CTA a cada 300-400 palavras

**Problema: Baixa taxa de conversão**
- **Solução**: Testeheadline primeiro (80% do impacto)
- **Solução**: Simplifique oferta (muitas opções confundem)
- **Solução**: Fortaleça prova social (adicione depoimentos)

**Problema: Muitos cliques mas poucos agendamentos**
- **Solução**: revise CTA (promete demais?)
- **Solução**: qualifique no próprio anúncio
- **Solução**: adicione filtro ("para empresas com faturamento de...")

**Problema: Email com baixa taxa de abertura**
- **Solução**: Revise assunto (primeiras 40 caracteres críticas)
- **Solução**: Personalize nome no remetente
- **Solução**: Segmente lista (não envie para todos)

**Problema: Copy soa muito "venda"**
- **Solução**: Use mais CCDI e menos "compre agora"
- **Solução**: Adicione armadilhas antes da oferta
- **Solução: Mostre, não conte (demonstre valor)

## Frameworks Aplicados

### 1. CCDI em Copy

**Conectar:**
- Primeiros 20% da copy
- História de identificação
- "Se você está aqui, provavelmente..."

**Desapegar:**
- "Vou te ajudar a tomar uma decisão, seja sim ou não"
- "Não preciso te vender, preciso te ajudar a decidir"

**Descobrir:**
- Qual é o problema real?
- "Exatamente o que não está funcionando?"
- Diagnóstico profundo

**Intensificar:**
- Consequências de não agir
- "Como isso impacta sua vida/sono/relacionamentos?"
- Urgência interna real

### 2. Estrutura de LP High Ticket

**Headline:**
```
[opção A] Como [benefício] em [tempo] sem [dor]

[opção B] 7 Armadilhas que te impedem de [resultado]

[opção C] De [estado A] para [estado B] em [tempo]

[opção D] O Método [único] que [promessa]

[opção E] Psicólogos: Como chegar em R$50k/mês em 90 dias
```

**Problema (3 Armadilhas):**
```
Armadilha 1: [Nome] - [Descrição]
→ Por que falha: [Explicação lógica]

Armadilha 2: [Nome] - [Descrição]
→ Por que falha: [Explicação lógica]

Armadilha 3: [Nome] - [Descrição]
→ Por que falha: [Explicação lógica]
```

**Solução (Método Visualizado):**
```
Passo 1: [Nome do passo] - [O que faz]
Passo 2: [Nome do passo] - [O que faz]
Passo 3: [Nome do passo] - [O que faz]
Passo 4: [Nome do passo] - [O que faz]
Passo 5: [Nome do passo] - [O que faz]
```

**Prova Social (3 Cases):**
```
Case 1: [Nome] - [Resultado] em [Tempo]
"Depoimento curto e impactante"

Case 2: [Nome] - [Resultado] em [Tempo]
"Depoimento curto e impactante"

Case 3: [Nome] - [Resultado] em [Tempo]
"Depoimento curto e impactante"
```

**Oferta:**
```
O que você recebe:
- [Entregável 1]
- [Entregável 2]
- [Entregável 3]
- [Bônus exclusivo]

Investimento: R$ X ou Y× de R$ Z
Garantia: X dias incondicional
```

### 3. Headlines que Convertem

**Modelo "Como... Sem..."**
```
"Como vender R$100k/mês sem depender de indicações"
```

**Modelo "O Segredo de..."**
```
"O segredo que top performers usam para fechar R$50k+
(não é o que você pensa)"
```

**Modelo "7 Erros..."**
```
"7 erros que te fazem perder 80% das vendas high ticket"
```

**Modelo "De [Estado A] para [Estado B]"**
```
"De consultora 'amiga' para especialista premium em 90 dias"
```

**Modelo "Por Que..."**
```
"Por que seus clientes te pagam pouco (e o que fazer)"
```

### 4. Fórmula P.A.S. (Copy Clássica)

**P (Problem):**
```
Você está [estado desagradável]?
```

**A (Agitate):**
```
A cada dia que passa você [consequência negativa].
E pior: [outra consequência].
```

**S (Solution):**
```
O Método [nome] te ajuda [benefício] através de [processo].
```
