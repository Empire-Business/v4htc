# Workflow Iniciante: Escrevendo Copy para High Ticket

## Objetivo
Aprender a escrever copy persuasiva para ofertas High Ticket (R$2k-10k).

## Tempo Estimado
4-6 semanas de prática

---

## Semana 1: Fundamentos de Copy

### Dia 1-2: Entenda os Frameworks

Estude os 3 frameworks principais:
1. **PAS** (Problem-Agitate-Solve)
2. **AIDA** (Attention-Interest-Desire-Action)
3. **BAB** (Before-After-Bridge)

**Exercício:** Para cada framework, escreva 3 exemplos do seu nicho.

### Dia 3-4: Pratique Headlines

Use estas fórmulas:
- "How to [result] in [timeframe]"
- "The [number]-step [framework] for [result]"
- "From [before] to [after] in [time]"

**Exercício:** Escreva 20 headlines para sua oferta.

### Dia 5-7: Pratique Bullets

Crie 3 tipos de bullets:
1. How-to: "How to..."
2. Secret: "The secret to..."
3. Curiosity: "Why most [people]..."

**Exercício:** Escreva 50 bullets para sua oferta.

---

## Semana 2: Copy para Landing Pages

### Landing Page Structure

1. **Hero Section**
   - Headline
   - Subheadline
   - 3-5 bullets
   - CTA
   - Form

**Template:**
```
HEADLINE: Get [Result] in [Time] with [Method]

SUBHEAD: The exact [framework] that helped [number] 
of [people] achieve [outcome]

BULLETS:
✓ [Benefit 1]
✓ [Benefit 2]
✓ [Benefit 3]

CTA: GET INSTANT ACCESS
FORM: [Name] [Email]
```

### Exercício da Semana

Escreva uma landing page completa para sua oferta:
- Hero section
- Problem section (2-3 parágrafos)
- Solution section (seu método)
- Social proof (depoimentos placeholders)
- CTA

---

## Semana 3: Email Copy

### Estrutura de Email High Converting

1. **Subject Line** (deve ser aberto)
2. **Opening Hook** (primeiras 2 linhas)
3. **Value/Story** (corpo do email)
4. **CTA** (call to action)
5. **P.S.** (muitas pessoas lêem isso primeiro)

### Exemplos de Subject Lines

**Curiosity:**
- "The mistake that's costing you R$[k]k/year"
- "What nobody tells you about [topic]"

**Benefit:**
- "How to [result] in [time]"
- "The framework for [desired outcome]"

**Urgency:**
- "24 hours left [offer details]"
- "Last chance [consequence]"

### Exercício da Semana

Escreva sequência de 5 emails:
1. Delivery (lead magnet)
2. Story
3. Value tip
4. Framework intro
5. Soft offer

## Semana 4: VSL Script

### Estrutura de VSL (20-30 min)

**0-3 min: Hook**
```
"If you're [situation] and want [desire], this is the 
most important video you'll watch.

Here's why: Most [people] struggle with [problem] 
because [misconception].

I'm going to show you [transformation] in [timeframe]."
```

**3-8 min: Problem**
```
"You know you're [situation] when...

[Describe painful situation]

The cost of staying here is [consequence]."
```

**8-15 min: Solution**
```
"But there's a better way.

It's called [FRAMEWORK NAME].

Here are the components:
1. [Component 1]
2. [Component 2]
3. [Component 3]"
```

**15-20 min: Proof**
```
"Now you might be thinking, 'Does this work?'

Let me show you...

[Case study 1 with numbers]
[Case study 2 with numbers]"
```

**20-25 min: Offer**
```
"Here's what I have for you:

[Detailed breakdown with value stack]"
```

**25-30 min: Close**
```
"You have to act now because [scarcity].

[Guarantee]

[Final CTA]"
```

### Exercício da Semana

Escreva script VSL completo para sua oferta.

## Semana 5: Sales Page Copy

### Long-Form Sales Page Structure

1. **Headline** (transformação específica)
2. **Problem** (agite a dor)
3. **Solution** (seu método único)
4. **About** (credibilidade)
5. **What You Get** (entregáveis detalhados)
6. **Testimonials** (5-10 com números)
7. **FAQ** (5-10 objeções)
8. **Bonus** (bônus destacados)
9. **Guarantee** (garantia forte)
10. **Pricing** (stack de valor)
11. **Final CTA** (chamada à ação)

### Exercício da Semana

Escreva sales page longa para sua oferta (mínimo 2000 palavras).

## Semana 6: Revisão e Refinamento

### Checklist de Qualidade de Copy

**Headline:**
- [ ] Promessa específica
- [ ] Indica timeframe
- [ ] Menciona público
- [ ] Cria curiosidade

**Body:**
- [ ] Usa framework (PAS/AIDA/etc)
- [ ] Tem exemplos específicos
- [ ] Tem prova social
- [ ] Endereça objeções
- [ ] Benefícios, não features

**CTA:**
- [ ] Verbo de ação
- [ ] Instrução específica
- [ ] Uma única ação
- [ ] Cria urgência

### Exercício Final

Reveja todo o copy que escreveu nas últimas 5 semanas e:
1. Identifique pontos fracos
2. Reescreva seções fracas
3. Adicione mais bullets
4. Fortaleça CTAs
5. Adicione mais proof

## Prática Contínua

### Rotina Diária de Copywriter

1. **Ler bom copy** (30 min)
   - Sales pages de grandes marcas
   - Emails de grandes copywriters
   - VSLs de ofertas High Ticket

2. **Escrever copy** (60 min)
   - 3-5 headlines
   - 10-15 bullets
   - 1 email completo
   - Ou 1 seção de sales page

3. **Testar e A/B** (contínuo)
   - Nunca assuma que está perfeito
   - Sempre teste variantes
   - Deixe os dados decidirem

### Hábitos de Excelência

- **Colete exemplos** de copy que converteu
- **Estude psicologia** de persuasão
- **Teste tudo** assumptions
- **Reveja com frequência** (revisão mensal)
- **Peça feedback** de clientes e prospects

---

## Erros Comuns de Iniciantes

❌ **Evite:**
- Escrever para multidão (escreva para UMA pessoa)
- Usar jargão técnico
- Focar em features ao invés de benefícios
- Negligenciar prova social
- CTAs fracos ou vagos
- Copy genérico (pode ser para qualquer negócio)

✅ **Faça:**
- Escrever como se fosse para um amigo específico
- Usar linguagem simples e conversacional
- Leads with benefits
- Overdeliver na prova
- CTAs específicos e action-oriented
- Copy único e específico para sua oferta

---

## Métricas de Copy Eficaz

### Email:
- Open rate: 25-40% é bom
- Click rate: 3-8% é bom
- Reply rate: 1-5% é excelente

### Landing Page:
- Opt-in rate: 30-50% é bom
- Bounce rate: <50% é aceitável

### Sales Page:
- Read rate: 30-50% leem até o fim
- Conversion: 3-10% é bom para High Ticket

Se suas métricas estão abaixo disso, revise o copy.
