# Conceitos-Chave: Building High Ticket Funnels

## 1. The Trust Funnel Framework

Diferença fundamental entre Low Ticket e High Ticket:
- Low Ticket: 1-2 touchpoints → Compra
- High Ticket: 7-20+ touchpoints → Compra

High Ticket requer construção de TRUST antes de venda.

**O Funil de Confiança:**

```
COLD TRAFFIC
    ↓
LEAD MAGNET (Captura e primeiro valor)
    ↓
NURTURE SEQUENCE (7-20+ emails, valor consistente)
    ↓
TRIPWIRE/Webinar (Micro-commitment)
    ↓
CORE OFFER (High Ticket principal)
    ↓
HIGH TICKET UPSELL (Premium)
```

## 2. Os 5 Tipos de Funnis High Ticket

### 1. Webinar Funnel (R$3k-15k)
**Melhor para:** Ofertas com componente educacional

**Estrutura:**
1. Ads → Registro Webinar
2. Email Reminders (3x)
3. Webinar (45-90 min)
4. Offer Page (24-72h)
5. Follow-up Sequence (7-14 dias)

**Key Metrics:**
- Reg rate: 40-60%
- Show-up: 20-40%
- Attendee→Buyer: 5-15%

### 2. VSL Funnel (R$2k-8k)
**Melhor para:** Ofertas com componente visual forte

**Estrutura:**
1. Ads → Opt-in (lead magnet)
2. Thank You → VSL (15-45 min)
3. Email Sequence (3-5 vídeos/emails)
4. Sales Page (VSL extendido ou texto)
5. Application/Checkout

**Key Metrics:**
- Opt-in: 30-50%
- VSL completion: 40-70%
- Conversion: 2-8%

### 3. Application Funnel (R$10k-50k)
**Melhor para:** Ofertas que requerem qualificação

**Estrutura:**
1. Ads → Content (blog/video/social)
2. Lead Magnet (case study, framework)
3. Nurture (5-10 emails)
4. Application Page
5. Strategy Call
6. Proposal → Close

**Key Metrics:**
- Lead magnet conv: 30-50%
- App completion: 20-40%
- Call show-up: 60-80%
- Close rate: 20-40%

### 4. Challenge Funnel (R$2k-10k)
**Melhor para:** Audiência orientada para ação

**Estrutura:**
1. Ads → Challenge Registration (3-5 dias)
2. Welcome + Group Access
3. Daily Content (vídeo + action)
4. Live Q&A diário
5. Dia Final: Offer Reveal
6. Extended Access (pago)

**Key Metrics:**
- Reg rate: 20-40%
- Daily engagement: 30-60%
- Completion: 20-40%
- Conversion: 5-20%

### 5. Book/Podcast Funnel (R$5k-50k)
**Melhor para:** Ofertas baseadas em autoridade

**Estrutura:**
1. Ads → Free Book/Interview
2. Shipping Page (captura address)
3. Book Delivery
4. Nurture Sequence (reader's journey)
5. Upsell Sequence (digital → core → coaching)

**Key Metrics:**
- Claim rate: 60-80%
- Read rate: 30-50%
- Conversion: 3-10%

## 3. Page-by-Page Breakdown

### Opt-in / Lead Magnet Page

**Purpose:** Capturar contato em exchange de valor

**Essential Elements:**
1. Compelling headline (benefício focado)
2. Subheadline (reforça valor)
3. What they'll get (bullet points)
4. Who it's for (qualificação)
5. Form (campos mínimos)
6. CTA button
7. Social proof (below fold)

**Copy Framework:**
```
HEADLINE: Get [Result] in [Time] with [Method]

SUBHEADLINE: The exact [framework] that helped [number] 
of [clients] achieve [outcome]

WHAT YOU'LL DISCOVER:
✓ [Benefit 1]
✓ [Benefit 2]
✓ [Benefit 3]

WHO THIS IS FOR:
You want [outcome] if...
• [Situation 1]
• [Situation 2]
• [Situation 3]

GET INSTANT ACCESS ↓
[Form: Name, Email]
```

## 4. Email Sequences

### Welcome Sequence (New Leads)
**Duration:** 5-7 emails over 7-10 days
**Goal:** Build relationship, introduce framework, sell tripwire

**Email 1: Delivery (Immediate)**
- Deliver lead magnet
- Set expectations
- Invite to reply

**Email 2: Context/Story (Day 1)**
- Origin story
- Why this matters
- Empathy with struggle

**Email 3: Framework Intro (Day 2-3)**
- Introduce methodology
- Quick win tip
- Authority

**Email 4: Objection Handling (Day 4-5)**
- Address common objection
- Show alternatives failed
- Position solution

**Email 5: Tripwire Offer (Day 6-7)**
- Make low-ticket offer
- Show value
- Clear CTA

## 5. Conversion Optimization

### Key Metrics to Track

**Front-End:**
- Traffic source performance
- Opt-in rate (target: 30-50%)
- Landing page bounce rate
- Video completion rate
- Email open rate (target: 25-40%)
- Email click rate (target: 3-8%)

**Back-End:**
- Lead to customer rate (target: 3-10%)
- Cost per lead (CPL)
- Cost per acquisition (CPA)
- Customer lifetime value (LTV)
- Return on ad spend (ROAS)
- Average order value (AOV)

### A/B Testing Priorities

**High Impact:**
1. Headline (promise + specificity)
2. Lead magnet (type + delivery)
3. CTA button (copy + color)
4. Video length (short vs long)
5. Offer structure (price, bonuses)
6. Guarantee strength

## 7. Common Mistakes

❌ **Don't:**
- Skip trust-building for High Ticket
- Have too many steps (complexity kills)
- Make CTA unclear
- Ignore mobile (50%+ traffic)
- Forget follow-up
- Set and forget (optimize continuously)

✅ **Do:**
- Build trust at every stage
- Keep journey simple (max 5-7 steps)
- Have one clear CTA per page
- Optimize for mobile first
- Follow up relentlessly with value
- Test and improve constantly
