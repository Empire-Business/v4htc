# Conceitos-Chave: Objection Handling para High Ticket

## 1. Filosofia: Objections são Buying Signals

Objections não são rejeições - são requests por mais informação.

**Framework:**
1. Acknowledge - Valide concern
2. Isolate - Ensure é a única barreira
3. Reframe - Shift perspective
4. Respond - Provide new info
5. Check - Confirm resolved

## 2. As 10 Objeções Mais Comuns High Ticket

### 1. "It's too expensive"
**Reality:** They have money, não estão convinced de value

**Response:**
```
"What's the cost of NOT solving this?

If you stay where you are for 6-12 months, 
what will that cost you?"

"Compare to the investment, does it feel 
more like expense ou investment in your future?"
```

### 2. "I need to think about it"
**Reality:** Need mais info ou confidence

**Response:**
```
"Totally understand. What specifically would 
you like to think about?

The people who get best results make clear 
decisions and commit fully.

Does waiting get you closer to your goal?"
```

### 3. "I can probably figure this out myself"
**Reality:** They could, but at cost (time, money, frustration)

**Response:**
```
"You absolutely could.

But you've already spent [time] trying.

How's that working?

This is the shortcut I wish I had.

What's your time worth to you?"
```

### 4. "I don't have time"
**Reality:** Overwhelmed, not uninterested

**Response:**
```
"My most successful clients are BUSIEST people.

This is about working smarter, not harder.

[Time commitment] per week. Plus you'll 
SAVE time once implemented."
```

### 5. "I need to talk to my spouse/partner"
**Reality:** May need approval OR using as excuse

**Response:**
```
"I actually encourage that.

I'll send you a summary document for you two 
to review together.

Would you like me to hop on a quick call 
with both of you?"
```

### 6. "How do I know this will work for me?"
**Reality:** Lack confidence em outcome

**Response:**
```
"Here's how you know:

1. [Specific element relevant]
2. [Another element]
3. Case study: [Similar client story]

The system works. The question is: 
Will you work the system?"
```

### 7. "I've tried everything and nothing works"
**Reality:** Burned out e skeptical

**Response:**
```
"I hear the frustration.

Here's why those didn't work 
(and why this is different):

[Differentiation]

And if this doesn't work, 
[guarantee details]"
```

### 8. "Now's not the right time"
**Reality:** Never is "right time"

**Response:**
```
"Here's what I've seen:

1. People waiting for "right time" 
   (still waiting)

2. People who decide now 
   (creating their own timing)

The cost of waiting is [specific cost]."
```

### 9. "I want to finish [other program] first"
**Reality:** May be overwhelmed OR excuse

**Response:**
```
"I respect finishing what you start.

That said, are you stalled or making progress?

If stalled, maybe this is actually what you need.

If progressing, finish first and come back."
```

### 10. "Can you give me a discount?"
**Reality:** See value mas want negotiate

**Response:**
```
"I price based on value delivered.

Here's what I CAN do:
1. Payment plan
2. [BONUS for commitment now]
3. Payment terms that work

I can't discount the price, but I can 
make it workable. Which option?"
```

## 3. Advanced Techniques

### Feel, Felt, Found
```
"I understand how you feel.
[Name] felt the same way.
But here's what they found: [result]

Does that help?"
```

### The Question Return
```
"It's too expensive."
→ "Expensive compared to what?"

"I don't have time."
→ "If you had 2 extra hours/day, 
   what would you do?"

"I'm not sure this will work."
→ "What would need to be true?"
```

### The Empathy Bridge
```
"I hear you, and I want you to know: 
Your concern is valid.

[Validation]

Here's what I've learned: [Reframe]

Let me share: [Response]

Does that perspective help?"
```
