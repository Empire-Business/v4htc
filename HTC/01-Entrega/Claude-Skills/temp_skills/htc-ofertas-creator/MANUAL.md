# MANUAL DE USO - htc-ofertas-creator

## Sobre Esta Skill

A **htc-ofertas-creator** é sua especialista em criar ofertas High Ticket irresistíveis (R$2k a R$50k). Ela usa frameworks comprovados para construir ofertas que convertem leads frios em clientes comprometidos.

## O Que Esta Skill Faz

### Framework Aplicado: 7 Elementos da Oferta Perfeita

Cada oferta criada por esta skill inclui obrigatoriamente:

1. **Promessa de Transformação** - O resultado específico e mensurável
2. **O Mecanismo** - Seu método/framework único
3. **O Stack** - Todos os entregáveis detalhados
4. **Prova Social** - Testemunhos e cases de sucesso
5. **Reversão de Risco** - Garantia forte
6. **Escassez e Urgência** - Razões para agir agora
7. **Investimento e Opções** - Preço e formas de pagamento

## Como Usar

### Sintaxe Básica

```
/htc-ofertas-creator [solicitação]
```

### Exemplos de Uso

**Criar oferta do zero:**
```
/htc-ofertas-creator Criar oferta completa para programa de 12 semanas de business coaching. Foco em empresários de serviço com faturamento R$20k-100k/mês. Incluir 12 sessões individuais, material de apoio, e comunidade.
```

**Otimizar oferta existente:**
```
/htc-ofertas-creator Analisar e otimizar esta oferta:
[cole sua oferta atual]

Identificar pontos fracos e sugerir melhorias.
```

**Criar tiers de precificação:**
```
/htc-ofertas-creator Criar 3 níveis de oferta (Silver, Gold, Platinum) para programa de marketing digital. Silver deve ser entrada (~R$3k), Gold principal (~R$7k), Platinum premium (~R$15k).
```

**Calcular value stack:**
```
/htc-ofertas-creator Calcular value stack para:
- 12 módulos online
- 8 sessões de grupo
- 3 sessões 1:1
- Templates e frameworks
- Comunidade por 6 meses

Sugerir preço final.
```

## Perguntas Frequentes

**Q: A skill pode criar ofertas para qualquer nicho?**
A: Sim! A skill funciona para qualquer oferta High Ticket (R$2k-R$50k) em qualquer mercado: negócios, saúde, relacionamentos, carreira, etc.

**Q: E se eu não souber quanto cobrar?**
A: A skill calculará o preço ideal baseado em value stack e mercado. Você pode pedir apenas a estrutura de oferta sem precificação específica.

**Q: Posso pedir múltiplas variações?**
A: Sim! Peça "Crie 3 versões desta oferta com diferentes ângulos de promessa" ou "Crie 5 headlines diferentes para esta oferta".

**Q: A skill inclui copy completa?**
A: Para copy completa (landing pages, emails), use htc-copywriter. Esta skill foca na estrutura e elementos da oferta.

**Q: Como sei se a oferta está boa?**
A: A skill inclui checklist de qualidade. Você também pode pedir "Avalie esta oferta em escala de 1-10 e sugira melhorias".

## Dicas Profissionais

1. **Teste a oferta**: Antes de lançar, teste a promessa e o preço com 5-10 prospects qualificados
2. **Comece simples**: Não precisa criar todos os tiers de uma vez. Comece com uma oferta principal e adicione variações depois
3. **Guarantia forte**: Para High Ticket, garanta 30-60 dias incondicional. Isso remove objeções e demonstra confiança
4. **Bônus estratégicos**: Cada bônus deve eliminar uma objeção ou acelerar um resultado
5. **Escassez genuína**: Use escassez real (vagas limitadas, data de início) não artificial

## Solução de Problemas

**Problema: Oferta parece muito genérica**
- **Solução**: Seja mais específico sobre seu público e diferenciais
- **Solução**: Peça "Adicione elementos que tornem esta oferta única"

**Problema: Preço parece muito alto/baixo**
- **Solução**: Forneça contexto sobre seu mercado e clientes atuais
- **Solução**: Peça "Justifique o preço com casos de sucesso similares"

**Problema: Falta clareza na oferta**
- **Solução**: Peça "Simplifique a oferta para 3 pontos principais"
- **Solução**: Peça "Crie uma versão one-liner da promessa principal"
