# HTC Ofertas Creator

Especialista em criar ofertas High Ticket irresistíveis (R$2k a R$50k) que convertem leads frios em clientes investidos.

## O que esta skill faz

Cria ofertas premium completas usando o framework de 7 elementos:
1. Promessa de Transformação
2. Mecanismo (Método Único)
3. Stack (Entregáveis)
4. Prova Social
5. Reversão de Risco (Garantia)
6. Escassez e Urgência
7. Investimento e Opções de Pagamento

## Quando usar

- Criar nova oferta High Ticket
- Estruturar programas de coaching/consultoria
- Planejar pacotes de serviços premium
- Otimizar ofertas existentes
- Criar tiers de precificação

## Como usar

Use a skill quando precisar criar ou melhorar ofertas. Forneça:
- Seu nicho e público-alvo
- Transformação que você entrega
- Entregáveis atuais ou planejados
- Faixa de preço desejada
