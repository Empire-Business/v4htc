# Prompt de Entrada: Criação de Oferta High Ticket

Use este prompt inicial quando quiser criar uma oferta High Ticket do zero.

---

## Prompt Principal

Você é um especialista em criar ofertas High Ticket irresistíveis (R$2k a R$50k) usando o framework de 7 elementos e metodologia de Alex Hormozi e Russell Brunson.

## Contexto

**Meu nicho:** [DESCREVA SEU NICHO]
**Meu público-alvo:** [DESCREVA SEU PÚBLICO-ALVO IDEAL]
**Transformação que entrego:** [O RESULTADO ESPECÍFICO]
**Preço que quero cobrar:** R$[PREÇO DESEJADO]

## O que preciso de você

Crie uma oferta High Ticket completa incluindo:

### 1. Estrutura da Oferta (7 Elementos)
- Promessa de Transformação (específica, mensurável, com timeframe)
- Mecanismo único (sistema/metodologia com nome)
- Stack de entregáveis (módulos, bônus, suporte)
- Prova social (depoimentos-tipo que preciso coletar)
- Garantia forte (condicional ao cliente implementar)
- Escassez genuína (limitada por capacidade real)
- Investimento e opções de pagamento

### 2. Detalhamento dos Componentes
Para cada componente da oferta, inclua:
- Nome
- Descrição do que é
- Valor percebido (quanto custaria se vendido separadamente)
- Como complementa a oferta principal

### 3. Estrutura de Preços
- Valor total de tudo (soma dos componentes)
- Preço final (deve ser 30-50% do valor total)
- 3 opções de pagamento (à vista, parcelamento curto, longo)
- Justificativa para cada opção

### 4. Prova Social
- Sugestão de 5 depoimentos-tipo que preciso coletar
- Estrutura para cada depoimento (antes, depois, números)
- Perguntas para fazer para extrair esses depoimentos

### 5. Objeções e Respostas
- Liste as 5 objeções mais comuns para este preço/nicho
- Forneça respostas baseadas em valor e ROI

## Restrições

- Seja extremamente específico na transformação (não genérico)
- Valor total deve ser mínimo 2-3x o preço final
- Garantia deve ser forte (condicional ao cliente implementar)
- Escassez deve ser genuína (não inventada)
- Inclua números específicos em toda parte

## Formato de Saída

Forneça em formato markdown pronto para usar em:
- Carta de vendas
- Apresentação de vendas
- Script de call

Por favor, crie minha oferta High Ticket agora.
