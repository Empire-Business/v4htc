# Prompt de Refinamento: Polindo e Melhorando Sua Oferta

Use este prompt quando já tiver uma oferta criada mas quiser refiná-la, melhorar o copy ou ajustar elementos específicos.

---

## Prompt Principal

Você é um copywriter e estrategista de ofertas High Ticket especializado em polir e refinar ofertas para máxima conversão.

## Contexto

Aqui está minha oferta atual:

[COLE SUA OFERTA AQUI - estrutura completa]

## O que preciso refinar

Selecione UMA área abaixo para refinar:

### Opção A: Headline e Transformação
- Promessa muito vaga → mais específica
- Falta números → adicionar métricas
- Sem urgência → adicionar timeframe

### Opção B: Stack de Valor
- Valor total baixo vs preço → aumentar valor perceived
- Componentes mal descritos → detalhar benefícios
- Bônus genéricos → tornar estratégicos

### Opção C: Garantia
- Garantia fraca → fortalecer
- Sem condições → adicionar (protege ambos)
- Não comunicada → destacar na oferta

### Opção D: Prova Social
- Poucos depoimentos → adicionar mais
- Depoimentos genéricos → específicos com números
- Sem variedade → diversificar (texto, vídeo, caso)

### Opção E: Escassez e Urgência
- Escassez fraca → fortalecer com razão genuína
- Urgência ausente → adicionar deadline real
- Não crível → basear em fatos reais

## Para a área selecionada, forneça:

### 1. Versão Original
[O que está na sua oferta hoje]

### 2. Problema Identificado
Por que não está ótimo

### 3. Versão Refinada (Opção 1)
Primeira opção melhorada

### 4. Versão Refinada (Opção 2)
Segunda opção melhorada

### 5. Explicação das Mudanças
Por que cada mudança melhora a oferta

### 6. Teste A/B Sugerido
Como testar qual versão funciona melhor

## Formato de Saída

- Comparação lado a lado (antes/depois)
- Explicações claras
- Recomendação final

Por favor, refine [ÁREA SELECIONADA] da minha oferta agora.
