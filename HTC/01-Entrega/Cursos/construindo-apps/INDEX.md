# Construindo Apps para o Negócio com Claude Code
## Curso Completo para Empresários Sem Tempo

---

## Sobre Este Curso

Este curso é para **empresários sem conhecimento técnico** que querem construir seus próprios aplicativos e sistemas usando inteligência artificial.

**Não precisa saber programar. Não precisa entender de tecnologia.**

Você só precisa saber **o que quer** que o seu sistema faça.

---

## Os Dois Caminhos

| Claude Code | Lovable |
|------------|---------|
| Terminal | Navegador |
| Mais controle | Mais simples |
| Código no seu PC | Código na nuvem |
| Gratuito + GLM | Plano gratuito |
| Para quem quer mergulhar | Para quem quer rapidez |

**Ambos usam a mesma metodologia: PRD PRIMEIRO.**

## Metodologia: PRD PRIMEIRO

```
VOCÊ DESCREVE O QUE QUER (PRD)
           ↓
    ESCOLHE UMA FERRAMENTA
           ↓
   A IA CONSTRÓI SEGUINDO O PRD
           ↓
          TESTE
           ↓
          AJUSTE
           ↓
         PUBLICOU!
```

**O segredo:** Um bom PRD = bom resultado.

## Antes de Começar

Certifique-se de ter:
- [ ] Computador com acesso à internet
- [ ] Claude Code instalado (se escolher caminho 1)
- [ ] Conta no GitHub (gratuita)
- [ ] Conta no Vercel (gratuita)
- [ ] 1-2 horas por dia para dedicar ao projeto

**Dica:** Se ainda não tem o Claude Code instalado, faça o [Curso de Instalação](../instalacao-claude-code/INDEX.md) primeiro.
