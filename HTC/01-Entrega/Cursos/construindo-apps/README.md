## Metodologia: PRD-Primeiro

### Por Que PRD Primeiro?

**SEM PRD:**
```
"Quero um sistema de vendas"
→ IA cria algo genérico
→ Você: "Não é isso..."
→ Loop infinito
```

**COM PRD:**
```
Aqui está meu PRD detalhado
→ IA segue exatamente
→ Sistema funcional na primeira tentativa
```

### Como Fazer um Bom PRD

1. **Descrita o que é** em 1-2 frases
2. **Defina para quem** (usuários)
3. **Liste funcionalidades** com passo a passo
4. **Defina os dados** (o que vai guardar)
5. **Diga o que NÃO precisa** (MVP)
6. **Use o template:** [template-prd.md](./template-prd.md)
