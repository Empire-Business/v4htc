# HTC - Nova Estrutura de Entrega

## Visão Geral da Reorganização

A estrutura de entrega foi reorganizada para seguir o modelo **DEPARTAMENTO > PROJETOS** com separação clara entre **ESTRATÉGICO** (para o empresário) e **PRÁTICO** (para a equipe).

---

## Estrutura de Diretórios

```
HTC/01-Entrega/
├── Vendas/
│   ├── Projeto-Scripts-Vendas/      [ESTRATÉGICO + PRÁTICO]
│   ├── Projeto-Funnel-Closure/      [ESTRATÉGICO + PRÁTICO]
│   └── Projeto-Objecoes/            [ESTRATÉGICO + PRÁTICO]
│
├── Marketing/
│   ├── Projeto-Funnel-Marketing/    [ESTRATÉGICO + PRÁTICO]
│   ├── Projeto-Conteudo-Organico/   [ESTRATÉGICO + PRÁTICO]
│   └── Projeto-Criativos/           [ESTRATÉGICO + PRÁTICO]
│
└── Operacoes/
    ├── Projeto-Metodo/              [ESTRATÉGICO]
    ├── Projeto-Instalacao/          [PRÁTICO]
    ├── Projeto-Construcao/          [PRÁTICO]
    └── Projeto-Analytics/           [ESTRATÉGICO + PRÁTICO]
```

---

## Guia de Navegação Rápida

### Para o EMPRESÁRIO (Foco Estratégico)

Comece aqui:
1. **Operacoes > Projeto-Metodo** - Entenda o método completo
2. **Vendas > Projeto-Scripts-Vendas > GUIA-EMPRESARIO** - Veja o impacto
3. **Marketing > Projeto-Funnel-Marketing > GUIA-EMPRESARIO** - Entenda o funil

Foco: ROI, KPIs, estratégia, implementação, escala.

### Para a EQUIPE (Foco Prático)

Comece aqui:
1. **Vendas > Projeto-Scripts-Vendas > GUIA-EQUIPE** - Scripts prontos
2. **Marketing > Projeto-Funnel-Marketing > GUIA-EQUIPE** - Funil prático
3. **Operacoes > Projeto-Instalacao** - Configure ferramentas

Foco: Execução, scripts, checklists, ferramentas, day-to-day.

## Marcação de Conteúdo

### [ESTRATÉGICO]
Conteúdo focado em:
- Por que fazer (justificativa)
- Impacto no negócio (ROI)
- Estratégia e planejamento
- KPIs e métricas
- Tomada de decisão

**Público:** Empresários, líderes, decisores

### [PRÁTICO]
Conteúdo focado em:
- Como fazer (implementação)
- Scripts e templates
- Checklists e guias
- Ferramentas e técnicas
- Day-to-day operacional

**Público:** Equipe de execução, implementadores

## Como Começar

### Se você é EMPRESÁRIO:
1. Comece por `Operacoes/Projeto-Metodo/`
2. Depois vá para `Vendas/Projeto-Scripts-Vendas/GUIA-EMPRESARIO.md`
3. Implemente projeto por projeto

### Se você é EQUIPE:
1. Comece por `Vendas/Projeto-Scripts-Vendas/GUIA-EQUIPE.md`
2. Depois `Marketing/Projeto-Funnel-Marketing/GUIA-EQUIPE.md`
3. Use as skills do Claude Code

### Se você quer IMPLEMENTAR TECNOLOGIA:
1. Comece por `Operacoes/Projeto-Instalacao/`
2. Configure Claude Code
3. Instale as skills relevantes

## Checklists de Implementação

### Checklist Geral de Implementação

#### Fase 1: Fundamentos (Semana 1-2)
- [ ] Ler GUIA-EMPRESARIO de cada projeto
- [ ] Definir KPIs e métricas
- [ ] Escolher primeiro projeto a implementar
- [ ] Comprar/configurar ferramentas necessárias

#### Fase 2: Primeiro Projeto (Semana 3-6)
- [ ] Ler GUIA-EQUIPE do projeto escolhido
- [ ] Completar estudo do material [ESTRATÉGICO]
- [ ] Implementar checklists [PRÁTICO]
- [ ] Começar execução pilot

#### Fase 3: Otimização (Semana 7-10)
- [ ] Analisar métricas do piloto
- [ ] Ajustar baseado em dados
- [ ] Documentar aprendizados
- [ ] Escalar para equipe

#### Fase 4: Expansão (Semana 11+)
- [ ] Adicionar próximo projeto
- [ ] Repetir processo
- [ ] Integrar projetos
- [ ] Otimizar sistema completo
