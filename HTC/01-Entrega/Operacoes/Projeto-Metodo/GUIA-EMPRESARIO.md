# Projeto Método - Guia para o Empresário

## Visão Estratégica

### Objetivo do Projeto
Documentar e estruturar o método completo HTC (High Ticket Club) para transformar especialistas em vendedores de alto valor consistentes e escaláveis.

### Valor Para o Negócio

**Métricas de Impacto:**
- Ke = 9.8 (196x mais eficiente que mentoria típica)
- Baseado em 157 transcrições de vendas reais
- 22 átomos de decisão identificados
- 13 trilhas estruturadas

**Retorno Sobre Investimento:**
- Investimento: Tempo de estudo
- Retorno: Sistema completo validado
- Time to value: Imediato (estudo sequencial)

### Quando Implementar

**Sinais de que você precisa deste projeto:**
- Você não tem método documentado
- Sua equipe não sabe vender de forma consistente
- Você depende de "talento" ao invés de sistema
- Você quer escalar mas não sabe como
- Seu fechamento é imprevisível

### Frameworks Principais

**CCDI (Capture, Create, Deliver, Increase):**
```
CAPTURE: Atrair leads qualificados
CREATE: Criar conteúdo que engaja
DELIVER: Entregar valor antes de vender
INCREASE: Aumentar valor ao longo do tempo
```

**Grand Slam Offer (Hormozi):**
```
Value = (Dream Outcome × Perceived Likelihood) ÷
        (Time Delay × Effort & Sacrifice)
```

**Street Selling (4 Pilares):**
```
1. CONEXÃO: Rapport profundo
2. DIAGNÓSTICO: Entender o desafio
3. APRESENTAÇÃO: Mostrar a solução
4. FECHAMENTO: Pedido de decisão
```
