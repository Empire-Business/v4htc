# HTC Claude Skills - README Completo

## Visão Geral

Este repositório contém 10 Claude Skills profissionais criadas especificamente para o HTC (High Ticket Consulting). Cada skill foi desenvolvida para automatizar e otimizar processos críticos de negócios High Ticket (ofertas de R$2k a R$50k).

## Lista de Skills Disponíveis

### 1. htc-ofertas-creator.zip
**Propósito:** Criar ofertas High Ticket irresistíveis

**O que faz:**
- Cria ofertas completas usando o framework de 7 elementos
- Calcula value stacks e precificação estratégica
- Desenvolve bônus e garantias
- Cria ofertas em múltiplos níveis (Silver/Gold/Platinum)

**Quando usar:**
- Criando uma nova oferta High Ticket
- Otimizando ofertas existentes
- Estruturando programas de consulting
- Precificando serviços premium

**Comandos:**
```
/htc-ofertas-creator Criar oferta para programa de coaching de 12 semanas
/htc-ofertas-creator Otimizar esta oferta: [cole sua oferta]
/htc-ofertas-creator Criar 3 tiers para oferta de consultoria
```

## Como Instalar as Skills

### Méthodo 1: Via Claude Code (Recomendado)

1. Baixe o arquivo .zip da skill desejada
2. No Claude Code, vá em **Settings** → **Skills**
3. Clique em **Upload Skill**
4. Selecione o arquivo .zip
5. A skill estará disponível imediatamente

### Método 2: Instalação Manual

1. Descompacte o arquivo .zip
2. Copie a pasta da skill para:
   - **Windows:** `C:\Users\[SeuUsuario]\.claude\skills\`
   - **Mac/Linux:** `~/.claude/skills/`
3. Reinicie o Claude Code

### Método 3: Instalação por Projeto

Para usar a skill apenas em um projeto específico:

1. Descompacte o arquivo .zip
2. Copie a pasta para: `[SeuProjeto]/.claude/skills/`
3. A skill estará disponível apenas neste projeto

## Como Usar as Skills

### Sintaxe Básica

```
/[nome-da-skill] [instrução]
```

### Exemplos Práticos

**Criando uma oferta completa:**
```
/htc-ofertas-creator Criar oferta para programa de consultoria de negocios de 12 semanas, focado em empresários de serviço com faturamento de R$20k-R$100k/mês. Incluir value stack, bônus e garantia.
```

**Escrevendo copy de vendas:**
```
/htc-copywriter Escrever landing page para programa de emagrecimento de 8 semanas. Foco em mulheres 30-50 anos. Include headline, bullets, CTA, e seção de FAQ.
```

**Criando sequência de email:**
```
/htc-lead-nurturing Criar sequência de 5 emails para nutrir leads que baixaram o case study mas não compraram ainda. Programa de R$5.000.
```

## Melhores Práticas

### 1. Seja Específico
Quanto mais específica sua instrução, melhor o resultado.

**Ruim:**
```
/htc-ofertas-creator Criar uma oferta
```

**Bom:**
```
/htc-ofertas-creator Criar oferta para programa de coaching executivo de 16 semanas, focado em diretores de vendas, preço R$15.000, incluindo 12 sessões 1:1 e material de apoio.
```

### 2. Use Contexto
Forneça contexto relevante sobre sua situação.

**Exemplo:**
```
/htc-sales-script Criar script para call com empresário que já expressou interesse mas disse "preciso pensar sobre isso". Programa de R$25.000.
```

### 3. Combine Skills
Use múltiplas skills em conjunto para resultados melhores.

**Exemplo:**
```
/htc-ofertas-creator Criar oferta para programa de 8 semanas
→ /htc-funnel-builder Desenhar funil para esta oferta
→ /htc-copywriter Escrever copy para todas as páginas do funil
→ /htc-sales-script Criar script de vendas para esta oferta
```

### 4. Iteração
Não se contente com o primeiro resultado. Peça ajustes.

**Exemplo:**
```
/htc-copywriter Escrever headline para landing page
→ "Muito genérico. Tente focar mais em resultado específico"
→ "Melhor. Agora torne mais urgente"
```
