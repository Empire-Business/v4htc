## Scripts e Estruturas Práticas

### Script Completo de Call (Template)

**Fase 1: Conexão (5-10 min) [PRÁTICO]**
```
"Obrigado pelo tempo hoje, [Nome].

Antes de começar, quero garantir que respeito seu tempo.
Temos [X] minutos juntos - isso ainda funciona?

Ótimo. O que eu proponho:
1. Primeiro, conheço sua situação
2. Depois, compartilho o que pode ajudar
3. Decidimos se faz sentido

Sombrou bem?

Também, só para saber: trabalho com pessoas que acredito
que genuinamente posso ajudar. Se no final não fizer sentido,
eu te digo. Combina?"
```

**Fase 2: Diagnóstico (15-20 min) [PRÁTICO]**
```
"Me conta sobre sua situação atual com [desafio]..."

[Perguntas SPIN]
- Situação: "Como funciona hoje..."
- Problema: "O que mais frustra você..."
- Implicação: "Quanto isso custa por mês..."
- Need-Payoff: "Como seria se resolvesse..."

"Quanto você está disposto a investir para resolver isso?"
```

**Fase 3: Apresentação (10-15 min) [PRÁTICO]**
```
"Baseado em tudo que você compartilhou, aqui está o que recomendo:

[Nome do Programa] - [Duração]

Isso vai ajudar você a [resultado específico] através de:

[Componente 1]: Resolve [desafio] ao [como]
[Componente 2]: Significa [benefício]
[Componente 3]: Te dá [resultado]

Investimento: R$[preço] ou R$[parcela]

Se isso te ajuda a [resultado deles], quanto vale para você?

Mais especificamente: Você investe R$[preço] uma vez.
E nos próximos [tempo], você ganha [ROI].

Isso é um retorno de [X]x no investimento.

Isso parece uma troca justa?"
```

**Fase 4: Fechamento (5 min) [PRÁTICO]**
```
"Então, [Nome], baseado em tudo que discutimos:
- Seu [desafio]
- Seu [objetivo]
- Seu [timeline]

Isso parece o ajuste certo?

[SE SIM] "Ótimo! Estou animado em trabalhar com você.
Aqui são os próximos passos: [passos]

Isso funciona para você?"

[SE TALVEZ] "Questão justa. O que te ajudaria a se sentir confiante?"

[SE NÃO] "Totalmente fino. Posso perguntar o que não parece certo?"
```

## Matriz de Objeções [PRÁTICO]

### Objeção: "Preciso pensar sobre isso"
```
"Totalmente entendo. Esta é uma decisão importante.

Pergunta rápida: O que especificamente você gostaria de pensar?

[Esperar resposta]

[Abordar preocupação específica]

E só para saber: as pessoas que obtêm os melhores resultados
são aquelas que tomam uma decisão clara (sim ou não) e se
comprometem totalmente.

Esperar parece que vai te aproximar do seu objetivo de [resultado]?"
```

### Objeção: "É muito caro"
```
"Ouço isso. Deixe-me perguntar:

Qual é o custo de NÃO resolver isso?

Se você ficar onde está por outro [período], quanto isso está
custando em [dinheiro/tempo/stress]?

[Esperar resposta]

Então, quando você olha assim, [preço] parece um investimento
ou uma despesa?"
```

### Objeção: "Posso fazer isso sozinho"
```
"Você absolutamente pode. E respeito isso.

Deixe-me perguntar: Quanto tempo você tem tentado resolver isso?

[Esperar]

E onde você está agora vs. onde quer estar?

[Esperar]

Aqui está o que eu sei: Você pode descobrir sozinho.
A questão é quanto tempo, dinheiro e frustração você está
disposto a gastar?

Este é o atalho. É o que eu gostaria de ter quando estava
começando.

Quanto vale seu tempo para você?"
```

## Checklist de Implementação

### Preparação Antes da Call [PRÁTICO]
- [ ] Pesquisar perfil social do lead
- [ ] Revisar respostas do formulário/call

## Troubleshooting Comum

### Problema: Taxa de Show-up Baixa (<70%)
**Solução [PRÁTICO]:**
- Melhorar confirmação de call
- Enviar SMS/lembrete 1h antes
- Criar urgência genuína
- Qualificar melhor leads

### Problema: Muitos "Preciso Pensar"
**Solução [PRÁTICO]:**
- Melhorar fase de diagnóstico
- Construir mais urgência
- Criar escassez genuína
- Pedir decisão mais claramente

### Problema: Objeções de Preço
**Solução [ESTRATÉGICO]:**
- Construir mais valor antes
- Usar value stack
- Comparar com custo do problema
- Oferecer opções de pagamento
