**Métricas:** Opt-in rate, CPL (Custo por Lead)

**1. LinkedIn (Principal)**

- Episódios curtos (10-15 min) educativos
- Interviews com alunos de sucesso

### Cálculos de ROI

**CAC (Custo de Aquisição de Cliente):**
```
CAC = Investimento Total Marketing / Número de Clientes

Exemplo: R$ 30.000 / 20 clientes = R$ 1.500 CAC
```

**LTV (Lifetime Value):**
```
LTV = Valor do Plano × Retenção Média (meses)

Exemplo: R$ 25.000 × 12 meses = R$ 300.000 LTV
```

**ROI (Return on Investment):**
```
ROI = (LTV - CAC) / CAC × 100

Exemplo: (300.000 - 1.500) / 1.500 × 100 = 19.900%
```

## PRINCÍPIOS DO FUNIL HTC

### Framework DotCom Secrets

**1. Value Ladder**
```
Gratuito (Lead Magnet) → R$ 0
Auto-venda (VSL)       → R$ 10k-50k
Fechamento (Call)      → R$ 10k-50k
Retention (Anuidade)   → R$ 10k-50k/ano
Multiplicação          → Upgrade planos
```

**2. Perfect Webinar**
```
QUEBRA (3 min)      → "Você está perdendo R$100k/mês"
HISTÓRIA (7 min)    → "Como descobrimos o método"
CONTEÚDO (30 min)   → "Framework CCDI explicado"
CLOSE (10 min)      → "Oferta irresistível"
Q&A (variável)      → "Objeções tratadas"
```

**3. Epiphany Bridge**
```
DESEJO              → "Quero fechar R$100k/mês"
FRUSTRAÇÃO          → "Dependo de indicações"
DESCOBERTA          → "Método CCDI"
TRANSFORMAÇÃO       → "Sistema previsível"
RESULTADO           → "R$100k+/mês consistentes"
```

### Framework Grand Slam Offer (Hormozi)

**Value Equation:**
```
Value = (Dream Outcome × Perceived Likelihood) ÷ (Time Delay × Effort & Sacrifice)

Dream Outcome:          Fechar R$100k/mês
Perceived Likelihood:   Ke=9.8, 157 transcrições validadas
Time Delay:             60-90 dias para resultados
Effort & Sacrifice:     5h/semanais, método passo a passo
```
