## Documento Completo de Métricas

**Versão:** 1.0
**Data:** 9 de Fevereiro de 2026
**Objetivo:** Acompanhar performance do funil + Otimizar conversões
**Framework:** Data-Driven Marketing + Analytics

## MÉTRICAS POR ETAPA DO FUNIL

### ETAPA 1 - ATRAÇÃO (Topo do Funil)

#### Métricas Principais

**1. Tráfego Total**
```
Definição: Total de visitantes que chegam à squeeze page
Benchmark: 10.000-50.000 visitantes/mês (após escala)
Meta: 5.000 visitantes/mês (Mês 1)
Fórmula: Pageviews únicos (Google Analytics)

Target por Etapa:
Mês 1: 1.000 visitantes
Mês 3: 3.000 visitantes
Mês 6: 10.000 visitantes
```

**2. Opt-in Rate (Taxa de Captura)**
```
Definição: % de visitantes que viram leads
Benchmark: 30-50% (Excelente), 20-30% (Bom), <20% (Otimizar)
Meta: 35% opt-in rate
Fórmula: (Leads captados / Visitantes únicos) × 100

Otimização:
- Se < 20%: Testar headline + lead magnet
- Se 20-30%: Testar CTA + design
- Se 30-40%: Testar forms + campos
- Se 40-50%: Excelente! Escalar tráfego
```

**3. CPL (Custo Por Lead)**
```
Definição: Custo médio para capturar 1 lead
Benchmark: R$ 50-150 (LinkedIn Ads), R$ 20-80 (Google Ads)
Meta: R$ 75 CPL
Fórmula: Investimento em tráfego / Número de leads

Target por Canal:
LinkedIn Ads: R$ 50-100 CPL
Google Ads: R$ 30-80 CPL
YouTube Orgânico: R$ 10-30 CPL (tempo investido)
Podcast Guest: R$ 20-50 CPL
```

**4. Lead Quality Score**
```
Definição: Qualidade do lead baseada em engajamento
Parâmetros:
- Completou Quiz de Maturidade: [SIM/NÃO]
- Abriu e-mail de boas-vindas: [SIM/NÃO]
- Clicou em VSL: [SIM/NÃO]
- Agendou call: [SIM/NÃO]

Score: 0-100 (baseado em parâmetros acima)
Meta: Score médio de 70+

Segmentação:
- Lead Frio (Score 0-30): 30% do total
- Lead Morno (Score 31-69): 50% do total
- Lead Quente (Score 70-100): 20% do total
```

### ETAPA 2 - NUTRIÇÃO (Meio do Funil)

#### Métricas Principais

**1. Email Open Rate**
```
Definição: % de leads que abriram e-mail
Benchmark: 40-60% (Excelente), 25-40% (Bom), <25% (Otimizar)
Meta: 50% open rate
Fórmula: (Aberturas únicas / Enviados - Bounces) × 100

Otimização:
- Se < 25%: Repensar assunto + remetente
- Se 25-40%: Testar assuntos (A/B)
- Se 40-50%: Testar preview text
- Se 50-60%: Excelente!
```

**2. Email Click-Through Rate (CTR)**
```
Definição: % de leads que clicaram em link no e-mail
Benchmark: 5-10% (Excelente), 3-5% (Bom), <3% (Otimizar)
Meta: 7% CTR
Fórmula: (Cliques únicos / Aberturas únicas) × 100

Otimização:
- Se < 3%: Repensar copy + CTA
- Se 3-5%: Testar posicionamento do CTA
- Se 5-7%: Testar texto do link
- Se 7-10%: Excelente!
```

**3. Unsubscribe Rate**
```
Definição: % de leads que descada da lista
Benchmark: <2% (Excelente), 2-5% (Aceitável), >5% (Problema)
Meta: <1.5% unsubscribe
Fórmula: (Descadastros / Enviados) × 100

Alerta:
- Se > 3%: Rever frequência de envio
- Se > 5%: Rever relevância do conteúdo
```

**4. Engagement Score**
```
Definição: Pontuação baseada em interações com e-mails
Parâmetros:
- Abriu 3+ e-mails: 20 pontos
- Clicou 2+ links: 30 pontos
- Assistiu VSL: 40 pontos
- Respondeu e-mail: 10 pontos

Score: 0-100
Meta: Score médio de 60+

Segmentação:
- Engajado Baixo (0-39): 40% do total
- Engajado Médio (40-69): 45% do total
- Engajado Alto (70-100): 15% do total
```

### ETAPA 3 - CONVERSÃO (Fundo do Funil)

#### Métricas Principais

**1. VSL Completion Rate**
```
Definição: % de leads que assistiram VSL até o fim
Benchmark: 60-80% (Excelente), 40-60% (Bom), <40% (Otimizar)
Meta: 70% completion
Fórmula: (VSL completas / VSL iniciadas) × 100

Otimização:
- Se < 40%: Repensar hook + estrutura
- Se 40-60%: Testar duração (15-20 min)
- Se 60-70%: Testar CTA durante vídeo
- Se 70-80%: Excelente!
```

**2. VSL-to-Call Conversion Rate**
```
Definição: % de leads que agendaram call após VSL
Benchmark: 15-25% (Excelente), 10-15% (Bom), <10% (Otimizar)
Meta: 20% agendamento
Fórmula: (Calls agendadas / VSL completas) × 100

Otimização:
- Se < 10%: Repensar proposta de valor + CTA
- Se 10-15%: Testar urgência + escassez
- Se 15-20%: Testar bônus de decisão rápida
- Se 20-25%: Excelente!
```

**3. Direct Checkout Rate**
```
Definição: % de leads que compraram direto (sem call)
Benchmark: 3-8% (Excelente), 1-3% (Bom), <1% (Otimizar)
Meta: 5% checkout direto
Fórmula: (Compras diretas / Visitantes checkout) × 100

Observação:
Alguns leads preferem comprar direto. Não excluir essa opção.
```

**4. Time to Convert (TTC)**
```
Definição: Tempo médio desde opt-in até compra
Benchmark: 14-21 dias (Médio), 7-14 dias (Rápido), >21 dias (Lento)
Meta: 18 dias médios
Fórmula: Média de tempo de conversão por lead

Segmentação:
- Compradores Rápidos (0-7 dias): 20%
- Compradores Médios (8-21 dias): 60%
- Compradores Lentos (22+ dias): 20%
```

---

### ETAPA 4 - FECHAMENTO (Call)

#### Métricas Principais

**1. Call Show-Up Rate**
```
Definição: % de calls agendadas que realmente acontecem
Benchmark: 70-90% (Excelente), 50-70% (Bom), <50% (Problema)
Meta: 80% show-up
Fórmula: (Calls realizadas / Calls agendadas) × 100

Otimização:
- Se < 50%: Repensar qualificação
- Se 50-70%: Melhorar confirmação (SMS + WhatsApp)
- Se 70-80%: Enviar lembretes estratégicos
- Se 80-90%: Excelente!
```

**2. Call Closing Rate**
```
Definição: % de calls que resultaram em venda
Benchmark: 20-40% (Excelente), 15-20% (Bom), <15% (Otimizar)
Meta: 30% closing
Fórmula: (Vendas / Calls realizadas) × 100

Otimização:
- Se < 15%: Repensar script + qualificação
- Se 15-20%: Treinar objection handling
- Se 20-30%: Ajustar apresentação de oferta
- Se 30-40%: Excelente!
```

**3. Average Deal Size**
```
Definição: Ticket médio de venda por plano
Benchmark (Real):
- PRIME: R$ 10.000
- PRO: R$ 25.000
- ELITE: R$ 50.000

Média ponderada: R$ 20.000
Meta: Aumentar mix de PRO/ELITE para R$ 25.000 médios
Fórmula: (Total vendas / Número de vendas)

Estratégia:
- Incentivar upgrade na call
- Oferecer bônus para planos superiores
- Mostrar ROI de plano maior
```

### ETAPA 5 - PÓS-VENDA (Retenção)

#### Métricas Principais

**1. Activation Rate**
```
Definição: % de clientes que ativamente usam a plataforma
Benchmark: 60-80% (Excelente), 40-60% (Bom), <40% (Problema)
Meta: 75% ativos
Fórmula: (Clientes ativos / Clientes totais) × 100

Otimização:
- Se < 40%: Onboarding mais intensivo
- Se 40-60%: Engajamento constante (e-mails, WhatsApp)
- Se 60-80%: Excelente! Manter estratégia
```

**2. Retention Rate (Renovação)**
```
Definição: % de clientes que renovam anuidade
Benchmark: 80-90% (Excelente), 70-80% (Bom), <70% (Problema)
Meta: 85% renovação
Fórmula: (Renovações / Clientes elegíveis à renovação) × 100

Otimização:
- Se < 70%: Repensar valor entregue + suporte
- Se 70-80%: Oferecer benefícios de renovação
- Se 80-90%: Excelente!
```

**3. Upsell Rate**
```
Definição: % de clientes que fazem upgrade de plano
Benchmark: 15-25% (Excelente), 10-15% (Bom), <10% (Otimizar)
Meta: 20% upgrade
Fórmula: (Upgrades / Clientes elegíveis) × 100

Estratégia:
- Oferecer upgrade após primeira venda high ticket
- Oferecer upgrade após atingir meta de faturamento
- Oferecer upgrade próximo à renovação
```

**4. LTV (Lifetime Value)**
```
Definição: Valor total que cliente gera durante relacionamento
Benchmark:
- PRIME: R$ 20.000-40.000 (2-4 anos)
- PRO: R$ 75.000-150.000 (3-6 anos)
- ELITE: R$ 200.000-500.000 (4-10 anos)

Meta: Aumentar LTV médio para R$ 100.000
Fórmula: (Ticket médio × Anos de permanência × Renovação)

Estratégia:
- Aumentar retenção (anos de permanência)
- Aumentar upsell (ticket médio)
- Aumentar cross-sell (produtos adicionais)
```

## MÉTRICAS FINANCEIRAS

### CAC (Custo de Aquisição de Cliente)
```
Definição: Custo médio para adquirir 1 cliente pagante
Benchmark: R$ 1.500-3.000
Meta: R$ 2.000 CAC
Fórmula: Investimento total marketing / Número de clientes

Cálculo Detalhado:
Investimento Marketing: R$ 30.000/mês
Leads Captados: 400
Leads Qualificados: 200 (50%)
Calls Agendadas: 40 (20%)
Calls Realizadas: 32 (80%)
Vendas: 10 (31.25% closing rate)

CAC = R$ 30.000 / 10 = R$ 3.000

Meta: Otimizar para R$ 2.000:
- Aumentar opt-in rate (35% → 45%)
- Aumentar agendamento rate (20% → 25%)
- Aumentar closing rate (31% → 40%)
```

### ROI (Return on Investment)
```
Definição: Retorno sobre investimento em marketing
Benchmark: 300-500% (Excelente), 200-300% (Bom), <200% (Problema)
Meta: 400% ROI
Fórmula: ((Receita - Custo) / Custo) × 100

Cálculo:
Investimento Marketing: R$ 30.000
Receita Vendas: R$ 200.000 (10 clientes × R$ 20k médios)
Lucro: R$ 170.000

ROI = ((170.000 - 30.000) / 30.000) × 100 = 467%

Meta: Aumentar para 500%:
- Aumentar ticket médio (R$ 20k → R$ 25k)
- Aumentar volume de vendas (10 → 12 clientes)
- Reduzir CAC (R$ 3k → R$ 2k)
```

### Payback Period
```
Definição: Tempo para recuperar CAC
Benchmark: 1-3 meses (Excelente), 3-6 meses (Bom), >6 meses (Problema)
Meta: 2 meses payback
Fórmula: CAC / (Receita mensal por cliente)

Cálculo:
CAC: R$ 3.000
Ticket médio: R$ 20.000 (pagamento anual)
Receita mensal reconhecida: R$ 1.667

Payback = 3.000 / 1.667 = 1.8 meses

Meta: Reduzir para 1.5 meses:
- Oferecer pagamento mensal (fluxo de caixa)
- Aumentar ticket médio
```
