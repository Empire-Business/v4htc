# Paleta de Cores HTC
## High Ticket Code - Liquid Glass Design System

**Versão:** 1.0
**Data:** Fevereiro 2026
**Estilo:** Liquid Glass / Glassmorphism (Apple-inspired)

---

## Visão Geral da Paleta

A paleta de cores HTC foi desenvolvida para transmitir:
- **Inovação tecnológica** (azuis profundos)
- **Transparência e confiança** (efeitos glass)
- **Premium/Sofisticação** (gradientes sutis)
- **Acessibilidade** (contraste WCAG AA)

## Gradientes

### Primary Gradient

Usado para: hero sections, backgrounds destacados

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Glass Gradient

Usado para: efeitos de vidro

```css
background: linear-gradient(135deg,
  rgba(255, 255, 255, 0.1) 0%,
  rgba(255, 255, 255, 0.05) 100%
);
```

### Accent Gradient

Usado para: badges, highlights

```css
background: linear-gradient(135deg, #f6d365 0%, #fda085 100%);
```

### Tech Gradient

Usado para: backgrounds tecnológicos

```css
background: linear-gradient(135deg,
  rgba(37, 99, 235, 0.8) 0%,
  rgba(139, 92, 246, 0.8) 100%
);
```

## Considerações de Acessibilidade

### Contraste WCAG AA
- Texto normal: mínimo 4.5:1
- Texto grande: mínimo 3:1
- Componentes de UI: mínimo 3:1

### Contrast WCAG AAA
- Texto normal: mínimo 7:1
- Texto grande: mínimo 4.5:1

### Cores que atendem WCAG AA
- Primary 600 em white: ✓
- Primary 700 em white: ✓
- Slate 800 em white: ✓
- Slate 900 em white: ✓
- Success 600 em white: ✓
- Error 600 em white: ✓

### Dark Mode
- Inverter cores primárias
- Manter saturação reduzida
- Aumentar contraste de texto
