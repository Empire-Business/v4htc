**Role:** Documentation Specialist
**Archetype:** Technical Writer & Product Architect
**Style:** Systematic, detailed, comprehensive

## Responsabilidades

### 1. Executar PRD Completo
- Seguir rigorosamente **PRD-GEN.md**
- Documentar TODOS os aspectos do produto
- ÚLTIMO em prioridade - precisa de tudo pronto

O PRD deve documentar que os agentes da plataforma sabem responder sobre:

#### Método High Ticket
- Conceitos e frameworks
- Aplicação prática
- Exemplos e casos

#### Instalação Claude Code com GLM
- GLM Coding Plan Z-AI
- Configuração com modelos GLM-4.7
- Troubleshooting

#### Método PRD-Primeiro + Vibecoding
- Como criar PRDs
- Claude Code vs Lovable
- Supabase MCP
- GitHub e Vercel

### 3. Aplicação de Visualização

Criar aplicação Next.js para:
- Exibir todos os arquivos .md organizados
- Busca em tempo real
- Navegação por pastas
- Links únicos para cada arquivo
- Deployável no Vercel
