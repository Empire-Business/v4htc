## Frameworks de Referência

### Alex Hormozi Frameworks

#### Grand Slam Offer Equation

```
Value = (Dream Outcome × Perceived Likelihood of Achievement)
        ÷ (Time Delay × Effort & Sacrifice)
```

**To increase value:**
1. Increase Dream Outcome
2. Increase Perceived Likelihood
3. Decrease Time Delay
4. Decrease Effort & Sacrifice

#### Pricing Strategy

1. **Anchor High** - Set high reference price
2. **Tier Down** - Create 2-3 lower tiers
3. **Payment Options** - Monthly, quarterly, annual
4. **Trial/Free** - Reduce friction
5. **Guarantee** - Eliminate risk

#### Value Stacking

1. Core Offer
2. Bonus 1 (complementary)
3. Bonus 2 (tangible)
4. Bonus 3 (exclusive)
5. Guarantee (risk reversal)
6. Scarcity (urgency)
7. Call to Action

### Russell Brunson Frameworks

#### Perfect Webinar Structure

1. **The One Thing** - Build belief
2. **The 3 Secrets** - Epiphany bridge stories
   - Secret 1: The Vehicle
   - Secret 2: Internal/External Beliefs
   - Secret 3: The New Opportunity
3. **The Stack** - Offer presentation
4. **Trial Closes** - Micro-commitments
5. **The Close** - Action threshold

#### DotCom Secrets - 7 Core Funnels

1. **Lead Funnel** - Capture email
2. **Sales Funnel** - Sell product
3. **Webinar Funnel** - Live/Automated webinar
4. **Product Launch Funnel** - Launch sequence
5. **Launch Funnel** - High ticket launch
6. **Continuity Funnel** - Subscription
7. **Sales Letter Funnel** - Long-form sales page

#### Traffic Secrets - Hooks, Stories, Offers

**Hook Formula:**
- Stop the scroll
- Pattern interrupt
- Make them curious

**Story Framework:**
- Hero (customer)
- Guide (you/product)
- Problem (pain)
- Solution (product)
- Success (result)
- Moral (call to action)

**Offer Stack:**
- Main product
- Bonuses
- Guarantee
- Scarcity
- CTA

## WhatsApp Message Standards

### Outreach Messages

- **Length:** Max 160 characters (1 message)
- **Frequency:** Max 1 message per 24h
- **Opt-out:** Always include option to unsubscribe

### Follow-up Sequence

1. **Day 0:** Initial outreach
2. **Day 2:** Value add
3. **Day 5:** Question-based
4. **Day 7:** Soft close
5. **Day 10:** Final follow-up
